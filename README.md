Ayurveda Health Assistant

We built this chatbot to explore how traditional Ayurvedic knowledge can be combined with a simple web interface.

The idea was to create something that can help users understand basic health issues and get natural remedy suggestions in a clean and easy way.

---
 What our project does

* Lets users describe their symptoms
* Matches them with possible Ayurvedic conditions
* Suggests herbs, remedies, and lifestyle tips
* Provides a herbs library with benefits and uses
* Includes different health categories (digestive, mental health, immunity, etc.)

 Built With

* HTML
* CSS
* JavaScript (basic logic, no frameworks)

⚙️ Features

* Single-page app style navigation
* Symptom analyzer with dynamic results
* 100+ conditions and remedies (static dataset)
* Herbs library with filters and search
* Clean UI inspired by natural/ayurvedic theme

---
 📁 Structure

* `index.html` → Main file (contains everything)
* CSS + JS → Included inside the same file

 Disclaimer

This is not a medical tool.
All information is for educational purposes only.
Always consult a doctor for real health issues.

 Note on AI usage

Some help was taken from AI tools while building parts of the UI, structuring ideas, and refining the content.
The project was still assembled, modified, and understood manually.
